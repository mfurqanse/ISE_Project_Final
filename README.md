# Student Attendance Portal

A comprehensive student attendance and grade management system built with vanilla HTML, CSS, and JavaScript.

## Features

### Admin Portal
- User Management (Create, Edit, Delete users)
- Course Management (Manage courses and enrollments)
- System Settings Configuration
- Dashboard with statistics

### Teacher Portal
- Mark Attendance for students
- Upload and manage marks/grades
- View student reports and analytics
- Dashboard with course overview

### Student Portal
- View attendance records
- Check results and grades
- Download transcripts
- Dashboard with personal statistics
